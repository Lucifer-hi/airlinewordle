✈ How to play AirlineWordle

A mystery airline is chosen each day. You have 6 guesses to identify it.

After each guess, the row shows which of 5 attributes match the target airline:

Green = exact match for that attribute.

A new hint unlocks with each guess — continent first, then region, country, hub, alliance, carrier type, IATA, ICAO, founding year, and status.

Search by airline name, IATA code (e.g. LH), or ICAO code (e.g. DLH).
